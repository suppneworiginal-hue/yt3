# Генератор історій для YouTube

Streamlit додаток для генерації історій на основі YouTube субтитрів.

## Встановлення

1. Встановіть залежності:
```bash
pip install -r requirements.txt
```

## Запуск

Запустіть додаток командою:

```bash
streamlit run app.py
```

## Структура проєкту

```
.
├── app.py                      # Головний файл Streamlit додатку
├── core/
│   ├── __init__.py
│   ├── config.py              # Конфігурація та константи
│   └── utils.py               # Допоміжні функції
├── services/
│   ├── __init__.py
│   ├── prompts.py             # Завантаження та заповнення промптів
│   └── generators.py          # Генератори (stub функції)
├── requirements.txt
└── README.md
```

## Stage 1

На поточному етапі реалізовано:
- UI з українською мовою
- Завантаження промпт-шаблонів з файлів
- Логіка підстановки змінних у промпти
- Stub функції для майбутньої інтеграції з YouTube та LLM

## Примітки

- Промпт-шаблони завантажуються з `story_core_prompt.txt` та `prompt_story.txt` (розміщені в корені проєкту)
- Якщо файли не знайдено, використовуються шаблони за замовчуванням
- Всі функції генерації поки що є stubs і повертають порожні рядки
